Rangkaian Antena TA

1.	buat ulang antena utama

2. 	ukur BW dan RL

3. 	buat duplikasinya 3 buah antena dengan parameter konstruksi yang sama
	parameter bisa diukur: lama solder, suhu solder, banyak timah, ukuran tembaga, ukuran 	teflon, ukuran core
	parameter tidak bisa diukur: kerapihan solder, penggunaan flux

4.	ukur semua antena (BW, RL, Gain, Polaradiasi)
5.	perbandingan parameter antena utama dan duplikasi
	data parameter konstruksi
	analisis akibat

	pencapaian: parameter konstruksi yang melalui takaran dapat menjadi solusi duplikasi
	tidak tercapai: faktor kerapihan solder, penggunaan flux

6.	penambahan resin
	parameter bisa diukur: takaran resin, takaran catalis, cetakan
	parameter tidak bisa diukur: rongga udara, posisi antena

7.	ukur semua antena(BW, RL, Gain, Polaradiasi)

8.	perbandingan parameter resin dan tidak
	data parameter konstruksi
	analisis akibat

	pencapaian: penambahan resin sesuai takaran menjadi solusi duplikasi
	tidak tercapat: faktor rongga udara dan posisi antena


1. buat antena utama (saat konstruksi mencatat parameter seperti: lama solder, suhu solder, banyak timah, ukuran tembaga, dan lainnya.) hal ini untuk keperluan duplikasi
2. ukur bandwidth dan returnloss
3. buat duplikasi 3 buah dengan menyesuaikan dengan parameter konstruksi.
4. ukur semua antena (bandwidth, return loss, gain, pola radiasi)
5. bandingkan hasil parameter ukur antena utama dan duplikasi, masukan juga data parameter konstruksi untuk di jadikan analisis jika antena duplikasi sesuai, ke akuratan takaran bisa menjadi data yang kuat, namun jika tidak stabil hasilnya maka kerapihan solder dan penggunaan flux bisa menjadi alasan.
6. penambahan resin guna antena yang kokoh
7. ukur kembali semua antena (bandwidth...)
8. bandingkan kembali parameter masing masign antena sebelum resin dan sesudah resin. masukan juga data parameter konstruksi resin seperti takaran resin dan katalis untukdijadikan analisis jika hasil sesuai berarti data parameter resin dapat memperkuat argumen, jika tidak sesuai, faktor rongga udara dan posisi antena pada resin dapat menjadi alasan.
